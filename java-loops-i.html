Java Loops IEASY

Objective

In this challenge, we're going to use loops to help us do some simple math.

Given an integer. \(N\), print its first \(10\) multiples. Each multiple \(N \times i\) (where \(1 \lt i \lt 10\) should be printed on a new line in the form: N x i = result.

Input Format

A single integer, \(N\).

Constraints

  • \(2 \lt N \lt 20\)

Output Format

Print \(10\) lines of output; each line \(i\) (where \(1 \lt i \lt 10)) contains the \(result\) of (\N \times i\) in the form:

N x i = result
